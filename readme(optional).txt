How to run in react


Note:- If NPM and Node.js is installed, Then Move to Step-2.


Step-1: Install NPM and Node.js from https://nodejs.org/en/

Step-2: Install create-react-app by running this command in your terminal:
	C:\Users\Your Name>npm install -g create-react-app.

Step-3: Run this command to create a React application named myfirstreact.
	C:\Users\Your Name>npx create-react-app myfirstreact.

Step-4: Run this command to move to the myfirstreact directory:
	C:\Users\Your Name>cd myfirstreact.

Step-5: Install ReactRouter and axios here by by running this command in your terminal
	C:\Users\Your Name\myfirstreact>npm install react-router-dom
	C:\Users\Your Name\myfirstreact>npm install axios

Step-7: Copy all the file in C:\Users\Your Name\myfirstreact\src

Step-8: Run this command to run the React application myfirstreact:
	C:\Users\Your Name\myfirstreact>npm start

Step-9: A new browser window will pop up with your newly created React App! 
	If not, open your browser and type localhost:3000 in the address bar.
